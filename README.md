_A language model architecture capable of probing other language models for behaviors that it wasn't trained to probe._
- Code for my publication [Meta-Models: An Architecture for Decoding LLM Behaviors Through Interpreted Embeddings and Natural Language](https://arxiv.org/abs/2410.02472)
- Funding approved and provided by [Effective Altruism Funds](https://funds.effectivealtruism.org/)
